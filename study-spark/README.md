# Spark

## Installation
```
$ wget https://downloads.apache.org/spark/spark-3.0.1/spark-3.0.1-bin-hadoop2.7.tgz
$ tar -xvf ./spark-3.0.1-bin-hadoop2.7.tgz
$ ln -s ./spark-3.0.1-bin-hadoop2.7.tgz ./spark
```

## Startup and Shutdown
```
// startup
$ ./bin/start-all.sh

// shutdown 
$ ./bin/start-stop.sh
```
